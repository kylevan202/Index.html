# Responsive Web App

A modern, responsive web application built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly across all devices (mobile, tablet, desktop)
- **Modern UI**: Clean and professional user interface
- **Interactive Elements**: JavaScript-powered interactivity
- **Mobile First**: Optimized for mobile devices with graceful scaling

## Project Structure

```
Index.html/
├── index.html      # Main HTML file
├── styles.css      # Responsive CSS styles
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No additional dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/kylevan202/Index.html.git
```

2. Navigate to the project directory:
```bash
cd Index.html
```

3. Open `index.html` in your web browser:
```bash
open index.html
```

## Usage

The app includes:
- **Navigation Bar**: Easy navigation between sections
- **Hero Section**: Eye-catching welcome section
- **About Section**: Information about the service
- **Services Section**: Three service cards showcasing offerings
- **Contact Form**: Get in touch with the team

## Responsive Breakpoints

- **Mobile**: < 600px
- **Tablet**: 600px - 1200px
- **Desktop**: > 1200px

CSS media queries ensure the app adapts beautifully to any screen size.

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Responsive design and styling
- **JavaScript**: DOM manipulation and interactivity
